# 🌱 Projeto Agrinho — Agro Forte, Futuro Sustentável

Este é um projeto web educativo e interativo desenvolvido especialmente para o **Concurso Agrinho 2026**. O site tem como objetivo apresentar a força do agronegócio brasileiro, destacando a união indispensável entre tecnologia de ponta, produtividade e preservação ambiental para a construção de um futuro sustentável.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído do zero, utilizando as tecnologias fundamentais do desenvolvimento Front-End (*Vanilla Web Stack*):

* **HTML5:** Estruturação semântica de todo o conteúdo do site, garantindo acessibilidade e divisões claras por seções através de tags modernas.
* **CSS3 (`style.css`):** Estilização visual completa, layout responsivo para dispositivos móveis e computadores, transições e a identidade visual baseada no tema do agronegócio.
* **JavaScript (`main.js`):** Camada de lógica e interatividade do usuário, responsável pelo dinamismo do ecossistema da página.

---

## 🛠️ Recursos e Funcionalidades Implementadas

O site conta com uma interface rica e dinâmica (Single Page Application - SPA), trazendo os seguintes recursos técnicos:

* **Navegação Dinâmica (SPA):** Troca de abas/seções de forma instantânea através de funções JavaScript, sem a necessidade de recarregar a página.
* **Suporte a Multi-idiomas (Internacionalização):** Sistema preparado para tradução dinâmica do conteúdo entre **Português (PT)**, **Inglês (EN)** e **Espanhol (ES)** via atributos `data-translate`.
* **Alternador de Tema (Dark Mode / Light Mode):** Sistema de troca de tema visual para melhorar a experiência de leitura do usuário em ambientes claros ou escuros.
* **Tela de Carregamento (*Loading Screen*):** Uma animação inicial de transição (loader) para melhorar a experiência do usuário enquanto o site processa os elementos.
* **Quiz Interativo:** Um sistema de perguntas e respostas sobre o agronegócio com validação de respostas via JavaScript e exibição do resultado final.
* **Design Responsivo e Imagens Otimizadas:** Uso de técnicas de design fluido e carregamento inteligente de imagens (`loading="lazy"`), garantindo rapidez no carregamento.

---

## 📂 Estrutura das Seções do Site

1.  **Início (Home):** Apresentação do projeto com banners informativos e cartões de destaque sobre Produção, Tecnologia e Sustentabilidade.
2.  **Agricultura:** Panorama geral sobre a potência agrícola do Brasil e suas principais culturas (soja, milho, café, etc.).
3.  **Tecnologia:** Demonstração do impacto da transformação digital no campo (drones, sensores e IA).
4.  **Sustentabilidade:** Foco nas práticas ecológicas essenciais para o futuro, como rotação de culturas e economia de água.
5.  **Quiz:** Área de gamificação para testar os conhecimentos adquiridos pelos visitantes.
6.  **Créditos:** Seção de perfil do desenvolvedor, detalhando a autoria, escola e o propósito do projeto.
